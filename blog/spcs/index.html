SPCS: Stock Price Comparison Script

2023-09-16

I am writing this just to share that I have written one R script which let's you compare stock prices of multiple shares using a chart. The name of the script is SPCS (Stock Price Comparison Script). Now there is not much ground breaking in the script. However, It's ability to use quantmod library for more than one share and it's ability to plot chart from the resulting data is in my opinion a good start for a useful project.

I don't see much need to go deep about it on this page. Information about the usage of the script and the code will be added to the github page. Anyways, I expect to add much more to that. It's still a kind of beta.
